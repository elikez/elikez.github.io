## 创建广告活动简介

当您开始制定广告活动策略时，您应该明确您的目标是什么，以及如何衡量成功

### 广告活动效率(广告活动目标一)

广告活动效率将根据 ACOS 或 ROAS 进行衡量。

**广告投入产出比 (ACOS)**
您可以使用 ACOS 来衡量广告活动效率。ACOS 等于您的广告活动支出金额除以广告活动开展期间的总销售额。ACOS 越低，广告活动效率就越高。低ACOS 表示您的销售额较高，广告支出较低。 您将在广告报告解读课程中了解有关此指标的更多信息。

**广告支出回报率 (ROAS)**
ROAS 是衡量广告活动效率的另一种方法：广告活动期间的销售额除以广告活动支出金额。ROAS 是 ACOS 的倒数。因此，您的 ROAS 越高，您的广告活动效率就越高。

### 品牌认知度（广告活动目标二）

品牌认知度根据**展示次数（Impressions）**进行衡量**。**
展示次数(impressions)表示您的广告向顾客展示了多少次。

## 选择要推广的Asin

无论您的目标是什么，都可以考虑推广畅销且做好销售准备的 ASIN。

* **畅销商品** 通过推广高销量商品，可以确保高点击率 (CTR) 带来更多的销量，让您的商品始终成为顾客的首选。
* **零售准备就绪 ASIN** 除了推广畅销商品外，您还可以推广那些通过零售准备检查清单审核的其他优质ASIN。推广这些 ASIN 有助于提高品牌认知度，并可能提高整个商品系列的销量。

### 根据(广告活动)目标选择Asin

#### 广告活动效率

如果您的目标是广告活动效率（根据 ACOS 或 ROAS 衡量），则应避免推广销量低或不符合零售准备指南的商品。这些 ASIN 可能无法帮助您实现广告活动目标。
**注意：** 推广销量低或未做好零售准备的商品可能会导致 ACOS 提高，广告活动效率降低。

#### 品牌认知度

如果您对特定的广告活动或商品的目标是提高品牌认知度（根据展示次数衡量），请考虑推广新商品或销量较低的商品。

## 选择搜索广告类型

### 商品推广和品牌推广

**同时**使用品牌推广和商品推广能够确保您通过我们的产品服务获得最大收益。您也可以选择使用其中一种，具体取决于您要推广的商品以及您的目标。

#### **商品推广** 

通过购物搜索结果和商品详情页上非常显眼的广告位，您可以将商品推广给正在寻找类似商品的顾客，从而提高曝光量并潜在地提高销量。商品推广可以轻松设置：它们不需要任何图片或自定义文案，并且在一小时内即可上线。这意味着您可以立即开始获得展示次数。

#### 品牌推广

品牌推广有助于提高品牌整个目录或新商品系列的品牌认知度和展示次数。另外，它们能够增加品牌旗舰店的流量，提供身临其境的购物体验，从而帮助提高品牌互动度。

您可以创建一个富有吸引力的标题，选择在广告左侧显示的图片。使用品牌Logo可以有效加强品牌建设，促使顾客点击您的广告，从而扩大受众规模。   

**请记住，品牌推广适用于已加入亚马逊品牌注册计划的专业卖家。**

## 设置广告预算和持续时间

创建广告活动后，您需要设置预算来控制支出。您可以设置每日预算，**如果使用品牌推广，您可以设置生命周期预算。** 
使用每日选项，您可以设置一个您愿意在一个日历月内**每天**花费的最大金额。**品牌推广，使用生命周期预算从而设置您愿意为整个广告活动支出的最大金额。您的广告活动会在用完预算结束，或到在到达广告活动结束日期结束。**

### 每日预算与生命周期预算

您的广告活动预算决定您在搜索广告上的支出。**商品推广活动按平均每日预算运行**。**品牌推广活动可以按平均每日预算，或生命周期预算运行。**

#### 平均每日预算

平均每日预算是指您愿意在一个日历月的每一天平均支出的最大金额。**最低每日预算为 $1。**   
您的每日预算是按一个月计算的。这意味着，**到日历月底，您的支出不会超过您设置的每日预算乘以当月的天数。**  
**您单日支出可能低于每日预算，也可能比每日预算最高多出 10%。** 这让您能够从流量较高的日期中受益。

每日预算在一天内不是匀速的支出。这让您能够从流量较高的时段中受益。 
**当广告点击次数达到每日预算所能购买的点击次数最大量时，您当天的广告将会停止投放，直到第二天重新开始。** 这意味着，如果您在一天的前几个小时内获得足够大量的点击次数，用完了当日预算，那么在当天剩下的时间里，您将不会再获得任何展示次数。 
由于顾客一天到晚都在购物，因此必须设置足够高的每日预算，保证全天广告持续投放。

亚马逊建议每日运算至少为10美金。

#### 生命周期预算（仅限品牌推广）

生命周期预算是指您愿意为这个广告活动支付的最大金额。**最低生命周期预算为 $100。**

与每日预算一样，生命周期预算在一天内的支出速度也不一样。如果您为广告活动设置了结束日期，那么生命周期预算将除以广告活动的天数。您一天内的支出可能会超出或少于此金额，但**广告活动开展期间的总支出绝对不会超出生命周期预算。**

创建广告活动后，可以增加生命周期预算，但无法减少。

### 广告活动持续时间

您需要决定是连续投放广告活动还是设置一个结束日期。您可以考虑您是要一直推广您的商品，还是要按季节或在有限时间内推广您的商品，从而在这两个选项之间进行选择。 

您最多可以提前 4 个月安排广告活动。设置广告活动持续时间时，您有两个选项可以选择：持续投放或选择结束日期。

#### 持续投放广告活动

如果您希望全年推广商品并且不想错过展示或销售机会，则持续投放广告活动会非常有用。您可以随时暂停广告活动，并在稍后继续投放。

#### 选择结束日期

如果您在一年中的相关的时间内销售季节性商品，或在有限的时间内推广商品，则设置广告活动结束日期就非常有用。结束日期一过，您的广告活动就会过期。 
您可以在广告活动过期之前的任何时间延长结束日期。如果广告活动过期后，您希望再次投放，您可以复制它，启动一个新的广告活动。

## 关键词竞价

### 选择关键词

对于商品推广和品牌推广，您可以将关键词作为整体策略的一部分。
如果输入您自己的关键词，一定要考虑顾客购买和发现商品的方式。亚马逊上的顾客可能会使用不同的购物查询内容搜索同一商品。最好**使用一系列与您的商品相关的关键词**，以便能够触达相关顾客。
关键词分为两类：**品牌和品类**。每种类型的效果不同，一定要了解每种类型的优点和缺点。

#### 品牌关键词

**优点:** 品牌关键词可以覆盖**搜索您品牌**的顾客，让他们能够更轻松地在亚马逊搜索结果中发现您的广告。 

品牌关键词可以使搜索结果中具有**高度相关性**，因此能够实现**更高的转化率**。它们不仅可以帮助您吸引新顾客，还可以让您的现有受众第一个想到您。

**缺点:** 品牌关键词**搜索量低**，这就意味着**展示次数少**。

#### 品类关键词

品类关键词可以覆盖**搜索普通商品**的顾客，让他们能够更轻松地在搜索结果的广告位中发现您品牌的该商品。 
与品牌关键词相比，使用品类关键词可以获得**更多搜索量**，这意味着可以**增加展示次数**。 
但是，品类关键词的**效率可能低于**品牌关键词，从而导致 **ACOS 提高，ROAS 降低**。


### 关键词竞价

选择关键词后，下一步是设置关键词竞价。搜索广告采用基于竞拍的单次点击成本(CPC) 库存购买模式。对于参加竞拍的广告，我们将综合考量广告主的竞价以及广告与顾客的购物查询内容的相关性，以便对广告进行排名并将其展示给顾客。如果顾客点击了竞拍获胜者的广告，则他们将支付略高于第二高单次点击成本（CPC）竞价的金额。

#### 设置竞价

设置竞价时，您可以选择您愿意为一个点击支付的最高金额。

#### 默认竞价（仅适用于商品推广）

对于商品推广，您的默认竞价适用于所有点击，除非您为关键词设置不同的竞价。这将限定您在顾客搜索所选关键词后点击您的广告时要支付的最高金额。建议您将竞价设置为您愿意为每次点击支付的最高金额。日后，您可以随时更改您的竞价。

#### 建议竞价

您可以选择建议竞价，也可以设置自定义竞价。我们根据与您的广告类似的广告历史竞拍洞察信息计算建议竞价，旨在为您的投放提供最佳初始竞价。

### 单个关键词竞价

通过手动广告活动，可以为单个关键词设置竞价。可能需要给出不同的竞价，具体取决于关键词和匹配类型。随着关键词的限制性和竞争性增强，您可能需要为其出更高的价，才能看到效果。

#### 广告活动/匹配类型

从广泛匹配，短语匹配再到精准匹配类型，预计竞价会越来越高
![](https://hexoblogpic.oss-cn-beijing.aliyuncs.com/活动匹配.png)

#### 关键词类型

从品牌到品类关键词类型，预计竞价不断提高。
![](https://hexoblogpic.oss-cn-beijing.aliyuncs.com/关键词类型.png)


### 品牌推广竞价策略

品牌推广具有一项称为**自动竞价**的功能，它允许亚马逊自动优化不同广告位的广告投入产出比。如果选择自动竞价，即表示允许亚马逊针对搜索结果顶部以下的广告位进行竞价。您设置的关键词竞价适用于搜索结果顶部广告位，它代表其他广告位的最大值。然后，亚马逊可能会降低其他广告位的竞价，基于它们的转化率。自动竞价永远不会提高您的竞价。
您可以选择停用自动竞价，设置**自定义竞价调整**，将其他广告位的竞价降低或提高到最大 99%。您可以针对搜索结果顶部以下的广告位设置此调整。例如，对于搜索结果顶部以下的广告位，将 $5.00 的竞价下调 40% 后将变为 $3.00。

### 交错竞价

关键字和匹配类型之间的竞价是一种策略，可以帮助您优化每个关键字的支出。 通常，关键词的限制性或竞争性越强，您的竞价可能需要越高。如果您不确定设置多高的竞价，则不妨先使用交错竞价。
例如：一般最具体的关键词（精准匹配）竞价>稍具体的词组匹配和品类关键词> 广泛匹配和品牌关键词> 自动广告活动竞价。


## 商品推广策略

### 最大化覆盖范围(定向方法)

自动定向+手动定向。

### 关键词还是商品？（投放类型）

商品推广允许按**关键词、ASIN** 或**品类**定向广告。建议同时使用商品定向和关键词定向，以便最大限度地覆盖购物搜索结果页面和商品详情页。 

卖家可以将关键词定向和商品定向用在同一广告活动中，但不能用在同一广告组中。供应商在一个广告活动中只能使用一种定向方法。  

请注意，如果您将商品定向和关键词定向用在同一个广告活动中，因为我们的系统限制，您**自我竞价**，以及无法提高单次点击成本CPC。（不懂）

### 商品推广竞价功能（竞价策略）

为商品定向设置竞价时，建议您使用广告活动创建工具中提供的建议竞价金额。在广告活动开展几周后，您可以查看效果，调整您的竞价和预算。
有些特定竞价功能适用于商品推广活动。您可以从**三种竞价策略**中进行选择。请选择最适合您的广告活动目标的策略。此设置适用于您广告活动中的所有竞价。

#### 动态竞价-只降低

当您的广告不太可能带来销售时，亚马逊将实时调整您的竞价。 选择此策略后，亚马逊可能会将竞价**下调**多达 100%**。

#### 动态竞价-提高和降低

当您的广告很有可能带来销售时，亚马逊将**实时调整您的竞价**（最高可提高 100%），并在您的广告不太可能带来销售时降低您的竞价。

#### 固定竞价

如果您选择“固定竞价”策略，亚马逊将针对所有机会使用您的确切竞价，而不会根据转化可能性调整您的竞价。 

相比动态竞价策略，固定竞价策略可能会让您从广告支出中获得更多展示次数，但您获得的转化量可能较少。

### 如何根据广告位调整竞价(投放展示位)

#### 了解广告位

广告位是指您的广告可能会在亚马逊上展示的位置。目前，您可以针对三种广告位设置不同的竞价并查看商品推广广告的效果：**搜索结果顶部广告位**（首页）、**搜索结果的其余位置**和**商品页面**。 

- 搜索结果顶部（首页）是指在搜索结果首页顶行的商品推广广告位。 
- 搜索结果的其余位置是指在搜索结果的中间或底部的商品推广广告位，以及在搜索结果的第二页及以后的所有商品推广广告位。 
- 商品页面是指商品详情页上的商品推广广告位以及搜索结果之外的某些其他广告位，如“加入购物车”页面。

#### 根据广告位设置竞价

除了选择竞价策略（动态竞价或固定竞价）之外，您还可以根据广告位设置不同的竞价。您可以为搜索结果顶部（首页）和商品页面这两个广告位的基础竞价输入增长百分比（最高为 900%）。 

执行此操作后，当您的广告竞争这些广告位展示时，您的竞价将会提高指定的金额。您的基础竞价（设置竞价）将应用于“搜索结果的其余位置”。

#### 工作原理

您可以同时使用“根据广告位调整竞价”功能和选定的竞价策略来确定最终应用的竞价。假设您要推广一款手表，并且将针对关键词“运动手表”的竞价设置为 $1.00。 此外，假设您分别为“搜索结果顶部（首页）”和“商品页面”广告位设置了 50% 和 25% 的调整幅度，那么根据广告活动竞价策略，这就是您竞价的应用方式。
![](https://hexoblogpic.oss-cn-beijing.aliyuncs.com/20231220103249.png)

#### 与固定竞价一起使用

您可以同时使用“根据广告位调整竞价”和“固定竞价”竞价策略来使您的广告在某个广告位上集中展示。 

例如，假设您的目标是通过赢得在搜索结果顶部（首页）广告位上展示的机会来提高您商品的曝光度。为此，您可以为关键词或广告组设置一个较低的竞价，并为搜索结果顶部（首页）设置一个较高的调整竞价的幅度。这将使广告在竞争搜索结果顶部展示机会时更具竞争力，同时还限制了其他广告位上的展示量（因为您的基础竞价较低）。

## 批量操作和API（高效创建和管理新广告活动）

### 批量操作

创建和优化广告活动的一个选项是批量操作。您可以使用电子表格对广告活动进行大规模更改，而不是在“活动管理器”中逐项更改。这是管理多个广告活动、关键词和广告的有效方式。

#### 更改类型

您可以使用“批量操作”同时进行多项更改，而不用在广告活动控制面板中逐一更改。这些更改包括： 

- 暂停广告活动
- 更新广告活动结束日期
- 更新每日预算
- 添加新的推广商品
- 暂停现有推广商品
- 添加新关键词
- 暂停现有关键词
- 更新关键词竞价

#### 批量操作的好处

- 可见性，可让您判断广告活动预算是否有余，或者广告活动预算是否提前用完
- 1 天、7 天、14 天、30 天和 60 天的广告活动、关键词和商品级别的业绩数据

### API

开展多个搜索广告活动的客户发现，通过 API 访问广告活动业绩数据，可以更轻松地管理广告活动。借助 API，开发人员可以构建自定义应用程序，用于控制亚马逊广告平台之外的广告活动。

#### 使用API

通过 API，您不仅可以设计用于创建新广告活动的应用程序，
还可以批量编辑和优化广告活动，从而帮助您实现业务目标并更有效地扩大广告策略的应用规模。

#### 管理广告活动

您可以通过两种方式来使用 API 管理广告活动：

- 您可以在组织内部构建工具，以便与 API 集成，这样可以获得您寻求的视图和功能。
- 您可以与第三方 API 集成提供商合作购买工具。有关我们所有 API 集成商的最新列表，请与您的业务代表联系。
